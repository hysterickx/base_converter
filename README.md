# Base Converter

A lightweight desktop application that converts numbers between different numeral systems (Binary, Octal, Decimal, and Hexadecimal). Built as a personal project to practice algorithm implementation, input filtering, and interactive GUI development using Python.

## 🚀 Key Features

- **Multi-System Conversion:** Seamlessly convert any number between Base 2, Base 8, Base 10, and Base 16.
- **Bi-directional Routing:** Choose any system as the source and any other system as the target for flexible conversions.
- **Wizard-Style Navigation:** Step-by-step user experience utilizing "Back" and "Next" layout states.
- **Clean Responsive UI:** Built with custom-configured interactive radio buttons and dynamic theme parameters.

## 🛠 Tech Stack

- **Language:** Python 3.11+
- **GUI Framework:** CustomTkinter
- **Architecture:** Object-Oriented Programming (OOP) with mathematical radix conversion logic.
